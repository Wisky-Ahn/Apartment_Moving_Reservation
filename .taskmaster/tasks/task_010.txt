# Subtasks:
## 1. 네비게이션 바 '내 정보' 링크 라우팅 경로 수정 [done]
### Dependencies: None
### Description: 네비게이션 바 컴포넌트에서 '내 정보' 링크의 라우팅 경로를 확인하고 올바른 사용자 프로필 페이지 경로로 수정합니다.
### Details:
1. 네비게이션 바 컴포넌트(NavBar.js 또는 유사한 파일)에서 '내 정보' 링크를 찾습니다.
2. 현재 '/user'로 설정된 경로를 '/profile' 또는 '/user/profile'과 같은 올바른 경로로 변경합니다.
3. Link 컴포넌트의 href 속성을 수정하거나, 클릭 이벤트 핸들러에서 라우팅 로직을 수정합니다.
4. 필요한 경우 라우팅 관련 import 문을 업데이트합니다.
